#### Preparing the AWS Environment

In this lab, we will deploy the system in the **Asia Pacific (Singapore) region — ap-southeast-1**.

Before creating any AWS resources, sign in to the **AWS Management Console** and check the selected region in the top-right corner of the screen. The region should be set to:

**Asia Pacific (Singapore) — ap-southeast-1**
