Cognito can automatically name the user pool based on the application. If the name is incorrect, change it to: `energy-waste-user-pool`.

1. On the user pool page, choose: **Overview**.
2. Click: **Rename**.
3. Enter: `energy-waste-user-pool`.
4. Click: **Save changes**.
5. The Overview page should display **User pool name: energy-waste-user-pool** and **Region: ap-southeast-1**.
