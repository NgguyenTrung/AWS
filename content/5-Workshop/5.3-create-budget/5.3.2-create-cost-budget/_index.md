Follow these steps to set up your cost threshold configurations:

1. Click on the **Create budget** button.
2. If prompted by AWS to select a budget setup type, choose **Use a template (simplified)**.
3. Under the **Templates - new** section, select the **Monthly cost budget** template.
4. Proceed to configure the detailed budget parameters as follows:
   * **Budget name:** `energy-waste-project-budget`
   * **Enter your budgeted amount ($):** `30.00`

![Create Budget Configuration Form](/images/5-Workshop/5.3-create-budget/create-budget-form.png)
> *Figure 5.3.2a: The monthly budget configuration form setup.*

1. After verifying the input information, click **Create budget** at the bottom right to complete the process. The system will then display a successful creation alert:

![Budget Created Successfully Alert](/images/5-Workshop/5.3-create-budget/budget-created-successfully.png)
> *Figure 5.3.2b: The status display page after the system successfully activates the budget.*

You can monitor and manage your existing budget rules and tracking thresholds directly on the main dashboard.

![Active Budgets Dashboard Overview](/images/5-Workshop/5.3-create-budget/budget.png)
> *Figure 5.3.2c: Overview list of active account budgets.*
